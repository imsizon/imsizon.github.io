USTC-DV-七月日记

日, 03 1, 2010

古往今来，无论中外，文学作品里爱情都占有相当的分量，可是，爱情到底是什么呢？不同的人眼里有不同的爱情，李志说“爱情不过是生活的屁”，裴多菲说"Liberty, love! These two I need. For my love I will sacrifice life, for liberty I will sacrifice my love."，甚至就连无所不知的百度也给爱情做了一番解释。

无意中在科大数字DV中心看到今年DV大赛的作品《七月日记》，因为一本日记本而认识的男、女主角，屡战屡败、屡败屡战的男配角，被爱情伤害而沉溺于悲伤世界、拍出来的照片也充满忧伤气息的女配角，各自在片中演绎着自己对爱情的理解。虽然由于器材和演员素质的限制，片子有些粗糙，却依然不能减低它在我眼中的光彩。不止因为片中熟悉的校园、甜蜜的爱情，更多的是片中关于爱情、关于理想、关于生活意义的思考。

Firefox-portable版本下载及附件组件安装

四, 24 12, 2009

上文《本博留言自动填写昵称、email和网址的GM脚本》推出后受到各位同学的欢迎，可是有部分同学还在受着ie的毒害，没有享受到使用Firefox的快感，因此小跟班不辞劳苦，克服了一贯的懒惰状态，发出此文。

1、Firefox的安装。其实google和百度可以告诉你哪儿能下载安装包，但是，作为一个懒人，能不安装的软件为什么要安装呢？所以，小跟班推荐Firefox Portable版本，放到U盘里就可以到处使用了。下载地址在这里，第一个就是简体中文版的下载链接。下载的文件是一个exe安装包，正常安装就好了，其实应该是一个自解压文件，安装好之后拷贝到其他地方依然可以正常使用的。

2、默认安装好的Firefox相比Maxthon之类的加强版ie还是有差距滴，所以，来给它加一些组件吧。小跟班制作了一个收藏集，可以一次安装常用的组件，请使用Firefox打开传送门。打开页面之后可以看到小跟班平常使用的组件了，每个组件后面都有一个大大的绿色按钮“添加到Firefox”。

好吧，这下你可以回到上一篇文章继续折腾本博的GM脚本了，顺带可以享受Firefox给你带来的其他快感。

本博留言自动填写昵称、email和网址的GM脚本

四, 24 12, 2009

每次回复留言的时候都要重复输入自己的昵称和邮箱，既浪费体力，又减少了键盘的寿命，那有木有什么方法可以减去这个步骤呢？

鉴于目前访问本博的几位同学中使用Firefox的较多，所以爱偷懒的小跟班同学将目光瞄向了Greasemonkey（呃？这是啥？百度知道！），写了一个极其简单的GM脚本。想和小跟班一样偷懒的同学请跟着以下步骤操作：

1、安装Greasemonkey：打开Firefox，访问Greasemonkey的插件发布页，点击“添加到Firefox”，安装并且重启Firefox。重启之后在Firefox的菜单“工具”栏中应该已经出现“Greasemonkey”的选项了，如果开启了状态栏，在状态栏上也会出现一个猴子头的图标，这样就说明安装成功啦。当然，已经安装过的同学就跳过这一步吧。

2、点击“工具”-“Greasemonkey”-“新建用户脚本”，或者在状态栏的猴子头上右键选择“新建用户脚本”，会出现一个新建用户脚本的对话框，在“名字”一栏随便输入一个值确认即可。此时会提示你选择一个编辑器，选择系统自带的记事本即可，或者你常用的其他文本编辑器。选择之后Firefox就会调用你选择的编辑器打开刚刚创建的脚本，内容大概如下：

// ==UserScript==
// @name samplename
// @namespace http://www.sina.com.cn/
// @include http://www.sina.com.cn/*
// ==/UserScript==

3、将以下代码复制粘贴到编辑器中覆盖原先的内容，将其中的“yourname”、“your email”换成自己的昵称和email保存即可，如果有自己的网站的话将最后一行的“//”去掉，并且替换“you site url if you have one”为你自己的网站地址：

// ==UserScript==

// @namespace http://www.imsizon.com/post/
// @description imsizon.com留言时自动填写昵称、email和link
// @include http://www.imsizon.com/post/*
// ==/UserScript==
document.getElementById('author').value='yourname';
document.getElementById('email').value='your email';
//document.getElementById('url').value='you site url if you have one';

4、打开小跟班的博客随便进入一篇文章，看看下面的评论栏是不是已经自动填上了你的昵称、email等信息？（什么？没有？检查一下“工具”-“Greasemonkey”中“启用”是否勾上了。还不行的话，请参照以上步骤再来一次吧）
